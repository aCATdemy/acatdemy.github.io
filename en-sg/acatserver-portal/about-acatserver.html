About aCATServer

Background:


aCATServer Network is a small, private Minecraft server hosted by aCATdemy.


The server started in 2018 as a test run when Mojang released the Village & Pillage Update (Minecraft 1.14). Since then, lessons were learnt and the server started to become more reliable. This translates to more people being invited to join the server.


History of Minecraft 1.14 - 1.15 Server:


As aforementioned, the server for Minecraft 1.14 to 1.15 was just a trial run, with players experiencing lags here and there due to the lack of knowledge of running a Minecraft server back then. This server maintained a healthy population for about three quarters of 2018 before shutting down as a result of time management issues.


History of Minecraft 1.16 Server:


In 2019, when Mojang released the Nether Update (Minecraft 1.16), the server started again with a brand new world. From the past experiences garnered, the server ran smoothly without any issues this time around. The server was shut down in April 2020 due to time management issues and low player counts.


2021 Server and Beyond:


Unlike the previous servers, there are no official names. In other words, the name changes with every server update. For future servers, it will be named "aCATServer Network" officially. In addition, a website for server members to view the latest and upcoming developments and patch notes for the server will also be made available.
