Knowledge Base

aCATServer Network Knowledge Base (KB) is a repository of logs made available for members of the aCATServer Network. It keeps a log of every issues or problem reported as the server runs. Each log bears a unique ID and description of the issue or problem. When a workaround or patch is available, the respective log will be updated to reflect it.


◉ LIVE: aCATServer Network Knowledge Base updates and refreshes every 60 seconds.
