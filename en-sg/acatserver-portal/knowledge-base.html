Knowledge Base

aCATServer Network Knowledge Base (KB) is a repository of issues discovered either by the server staff of members of the server. It keeps a record of every issue that arises as the server runs. Each record bears a unique ID and description of the issue. When a workaround/solution is available, the respective record will be updated to reflect it accordingly.

Please click on each record to expand it.


[SOLVED] KB1: Flying is not enabled on this server
Issue:

Members sometimes get disconnected/kicked from the server with the message, "Flying is not enabled on this server". This is due to the configuration of the server properties.


Solution (step by step):

1. Ensure that you have a stable internet connection. We recommend using an ethernet cable over a wireless connection.

2. Ensure that you are not using any mods, cheats, or unauthorised clients.

3. Run Windows Update to apply the latest patch and fixes.

4. If you use a third-party antivirus/firewall, either disable it or update its definition. aCATServer Network recommends Microsoft Defender and Windows in-built firewall.

5. Perform a disk cleanup.

6. If the issue persists after all these steps, perform a clean install of Minecraft.
