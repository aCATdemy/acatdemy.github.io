Java Uninstall Tool
Note:

Starting from Minecraft 1.12, installers come with their standalone version of Java by default.

aCATServer Network strongly recommends all members to remove any existing Java installations on their machine using the Java Uninstall Tool for security reasons.

This step is optional if you require Java for other purposes. However, practice cybersecurity responsibility by keeping applications, including Java, up to date to patch vulnerabilities and zero-day exploits.
